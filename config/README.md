#Cloud
###配置运行
> ###PHP环境配置（修改php.ini）
>>       建议环境PHP7+
>>       打开exif扩展
>>       打开com_dotnet扩展
>>       设置session.auto_start = 1
>>       设置post_max_size = 500M
> ###MYSQL环境配置（修改.ini）
>>      在my.ini 的[mysqld]下添加sql_mode=STRICT_TRANS_TABLES,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION
>>      还原cloud.sql文件
> ###Apache环境配置（修改.http.conf）
>>      打开LoadModule rewrite_module modules/mod_rewrite.so 如果前面有#去掉#
>>      设置 Options Index FollowSymLinks Includes ExecCGI 为 Options FollowSymLinks Includes ExecCGI
>###项目内环境配置
>>      修改service下server.php（socket服务）的$conf变量
>>      修改service\Controller下ServerController.php（数据服务）的$conf变量
>>      $conf=[
>>          "HOST"=>"127.0.0.1",                为数据库所在地址
>>          "PORT"=>"3306",                     为数据库端口
>>          "DATABASE"=>"cloud",                为数据库名
>>          "USERNAME"=>"root",                 为数据库用户名
>>          "PASSWORD"=>"123456",               为数据库密码  
>>          "ADDRESS"=>"D:\laragon\www\Cloud"   为项目绝对路径地址(Cloud文件所在地址)
>>      ];
###项目结构
>      public文件夹（静态资源文件，存放css、图片等资源）
>>      css（存放项目所需的外联样式表）
>>      font(字体文件)
>>      img(图片资源)
>>      js(js文件及Slimf.js框架)
>>      moudle(模块文件)
>      service文件夹（后台服务、php、控制器、路由、socket）
>>     Controller（存放控制器）
>>      mailer（邮箱服务）
>>      socket(socket服务)
>      upload文件夹（用户操作文件夹，用于存放默认数据及用户文件）
>>      group_head(群头像)
>>      user_headd(用户头像)
>>      user_bg(用户背景图片)
>      index.php 项目入口
>      main.php 项目主程序
>      share.php 项目分享功能
>      verify.php 用户注册验证服务
>      test.php 项目开发测试服务
###相关api
>###路由
>>      路由配置文件在service下的web.php
>>      路由写法为 'xx/test/(:any)'=>'Controller@function',
>>      xx/test为url部分后面对应控制器及控制器内的方法
>>      (:any)为路由参数，对应方法所需参数，可带多个(:any)为任何字符，(:num)为数字
>##socket服务
>>      socket服务文件在service下的server.php
>>      socket服务负责此项目的聊天部分，需要使用cli模式运行
>##控制器
>>     控制器在service下的Controller文件夹内
>>     新建控制器为名称+Controller
